# Managing Documents

Use this page to upload SDS (Safety Data Sheet), TDS (Technical Data Sheet), MSDS (Material Safety Data Sheet), or other supporting PDFs for the products you sell.

Go to **Admin → MDM → Documents**.

## 1. Find your product

When you open the page, you'll see a search screen with three filters:

| Filter | Example |
|---|---|
| Product Name | `Acetone` |
| SKU | `PRD-GRGNC-80500907` |
| CAS Number | `8050-09-07` |

Results only include products already linked to your seller account. Click a row to open its **Documents →** panel. If a product shows *"not available"* instead of a **Documents →** button, it isn't linked yet — contact the marketplace administrator.

## 2. Upload a document

With a product selected, click **+ Upload document** and fill in:

| Field | Notes |
|---|---|
| **PDF file** | Required. PDF only, max 20 MB. |
| **Document type** | SDS, TDS, MSDS, or Other. Required. |
| **Display name** | Optional — defaults to the uploaded file's name. |
| **Grade** | Optional free text, e.g. `Purity (GC) 99.5%`. |
| **Origin countries** | Optional — add one or more countries this document applies to. |

Click **Upload**. The document appears in the list once it succeeds.

## 3. Manage existing documents

For the selected product, the documents table shows type, name, grade, origin countries, file size, and upload date.

- **View PDF** — opens the file in a new tab.
- **Delete** — click Delete, then Confirm. You can only delete documents you uploaded yourself.
- Use the **Document Type** dropdown and **Search** box above the table to filter the list.

To work on a different product, click **← All products**.

## Notes

- Only PDF files are accepted; other formats are rejected before upload.
- The 20 MB limit is enforced in the browser before the file is sent.
- If you see *"This product is not available for documents"*, the product-to-seller link is missing on the MDM side — this isn't something you can fix from this page; contact the marketplace administrator.
